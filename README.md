# 🎂 Birthday Bot
A Discord bot to keep track of users' birthdays.

# 🤔 About

✅ Allow users to ste their own birthday.

✅ View the next and upcoming birthdays in the server.

✅ Shoutout users in specific channels and give them specific roles on their birthday.

✅ View anyones birthdays on the server.

## 📋 Usage

Want to add Milk Bot to your server? Add him [here](https://discord.com/api/oauth2/authorize?client_id=828325559402364969&permissions=116800&scope=bot)!

## 🖥️ Commands

- `/birthday <user>` - Displays a user's birthday. If it is not set, gives an option to ask them to set it.
- `/setbirthday <month> <day> <year>` - Sets your own birthday. Can only be done once every 365 days.
- `/next` - Shows the next upcoming birthday.
- `upcoming` - Shows the next 25 upcoming birthdays in order.
- `/setchannel` - **Owner Only** Sets the channel to send a message in on someone's birthday.
- `/setrole` - **Owner Only** Sets the role to give a user on someone's birthday.
